Car Image Classification using Convolutional Neural Network

Code Overview:

This HTML page provides a concise overview of a Python script for building and training a Convolutional Neural Network (CNN) to classify car images using the Keras library.

Code Components:

  • Image data preprocessing using Keras ImageDataGenerator for training, validation, and test sets.
  • Definition of a CNN model with convolutional and pooling layers, flattening, and dense layers.
  • Compilation of the model with the Adam optimizer and categorical crossentropy loss.
  • Training the model on the car image dataset for 10 epochs.

Directory Structure:

    - Validation/

    - Test/

Key Libraries Used:

Model Architecture:

Conv2D(32) -> MaxPooling2D -> Conv2D(64) -> MaxPooling2D -> Flatten -> Dense(128) -> Dense(10)

Training:

model.fit(train_generator, batch_size=32, epochs=10, validation_data=validation_generator, validation_steps=len(validation_generator) // 16)

Outcome:

Upon execution, the model is trained to classify car images into 10 classes with accuracy metrics tracked over 10 epochs.
